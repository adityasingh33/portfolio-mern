# Portfolio MERN Monorepo

This monorepo contains:

- `client`: Vite + React + TypeScript + Tailwind CSS
- `server`: Node + Express + TypeScript

Use workspaces to manage packages. Scripts are available at root to run common tasks.

## Quickstart

- Install: `npm install`
- Run dev: `npm run dev`
- Build: `npm run build`
- Preview (client): `npm run preview`

Environment variables: see `server/.env.example`.
