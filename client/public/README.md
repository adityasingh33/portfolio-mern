# Public Assets

This directory contains static assets that are served directly by Vite.

## Resume PDF

Place your resume PDF file here as `resume.pdf`.

**Instructions:**
1. Copy your resume PDF file to this directory
2. Name it exactly `resume.pdf`
3. The file will be accessible at `/resume.pdf` in your application

The "Download Resume" button in the header will link to this file, and the Resume page will also link to it.
